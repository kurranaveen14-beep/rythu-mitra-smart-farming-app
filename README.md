# Rythu Mitra – Smart Farming Assistant 🌱

Rythu Mitra is an AI-powered smart farming mobile application designed to help farmers manage crops efficiently.

## Features

- AI Crop Doctor – Upload crop images to detect diseases and get pesticide suggestions
- Weather Updates – Location-based weather forecast
- Agri Marketplace – Buy fertilizers and pesticides
- Farmer Knowledge Hub – Agricultural tips and guidance
- Government Schemes – Information about farmer benefits
- Bilingual Support – Telugu and English

## Technologies
- AI Image Analysis
- Mobile Application
- Geolocation Services

## Project Status
The application is available as an APK and is continuously being improved with additional AI features.
